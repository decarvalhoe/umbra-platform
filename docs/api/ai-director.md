# AI Director API

Base URL: `http://localhost:8001`

## Endpoints

### POST /api/v1/generate/quest
Generate an AI-powered quest based on player context.

### POST /api/v1/generate/dungeon
Generate a dungeon layout with rune cards and corruption.

### POST /api/v1/generate/narrative-event
Generate a narrative event with meaningful choices.

### POST /api/v1/director/evaluate
Evaluate a game session and recommend difficulty adjustments.

### POST /api/v1/director/recommend-content
Personalized content recommendations.

### GET /health
Health check.
