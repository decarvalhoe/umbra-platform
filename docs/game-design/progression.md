## 2. Affinité Compagnon (Issue #98)

### Système d’Affinité
Chaque compagnon possède un score d’**Affinité** allant de **0 à 100**, représentant la profondeur de la relation entre le joueur et le compagnon.

### Seuils d’Affinité

| Seuil | Nom | Affinité | Effets débloqués |
|-------|-----|----------|---------------------|
| I | **Inconnu(e)** | 0-19 | Dialogues basiques, compétence de combat T1 |
| II | **Familier/Familière** | 20-39 | Dialogues étendus, compétence T2, bonus idle +5% |
| III | **Ami(e) Proche** | 40-59 | Quêtes personnelles, compétence T3, bonus idle +10% |
| IV | **Intérêt Romantique** | 60-79 | Scènes intimes, compétence T4, bonus idle +15%, cadeau spécial |
| V | **Lien Profond** | 80-100 | Toutes les scènes, compétence ultime, bonus idle +25%, Void Form débloquable |

### Gagner de l’Affinité

| Action | Affinité gagnée | Limite |
|--------|-----------------|--------|
| Dialogue quotidien | +1-3 | 1/jour par compagnon |
| Cadeau apprécié | +3-8 | 3 cadeaux/jour (tous compagnons) |
| Run ensemble | +1-2 | Par run |
| Shadow Vigil | +0.5-1 | Par session (12h max) |
| Quête personnelle | +5-15 | Quand disponible |
| Choix narratif favorable | +2-10 | Contextuel |
| Cadeau détesté | -2-5 | Attention aux préférences! |

### Préférence Relationnelle
Le joueur peut définir sa **préférence relationnelle** pour chaque compagnon individuellement :

| Préférence | Effet |
|------------|-------|
| **Romance** | Scènes romantiques, dialogues amoureux, CGs intimes |
| **Amitié Profonde** | Scènes de camaraderie, dialogues chaleureux, CGs de confiance |
| **Neutre** | Dialogues professionnels, interactions minimales, pas de CGs spéciales |

La préférence peut être changée à tout moment sans perte d’affinité. Les scènes déjà vues restent dans la galerie.

## 3. Niveau de Résonance (Issue #104)

### Système de Résonance
Le **Niveau de Résonance** (1-15) représente le lien surnaturel entre le joueur et un compagnon. Il se débloque une fois l’Affinité ≥ 40 (Ami(e) Proche).

### Progression et Récompenses

| Niveau | XP Résonance requis | Récompense |
|--------|---------------------|-------------|
| L1 | 0 | Accès au système de Résonance |
| L2 | 100 | +2% stats du compagnon en combat |
| **L3** | 250 | **Echo Fragment Tier 1** (passif équipable) |
| L4 | 450 | +4% stats du compagnon |
| L5 | 700 | Scène narrative bonus |
| **L6** | 1000 | **Resonance Scene** (scène cinématique unique) |
| L7 | 1400 | +6% stats du compagnon |
| L8 | 1900 | Cosmetic exclusif pour le compagnon |
| L9 | 2500 | +8% stats du compagnon |
| **L10** | 3200 | **True Name** (le compagnon révèle son vrai nom, +10% stats) |
| L11 | 4000 | Titre joueur exclusif |
| **L12** | 5000 | **Echo Fragment Tier 2** (passif + actif) |
| L13 | 6200 | +12% stats du compagnon |
| L14 | 7600 | Aura visuelle spéciale |
| **L15** | 9200 | **Void Form** (transformation ultime du compagnon) |

### Gagner de l’XP Résonance

| Action | XP Résonance | Notes |
|--------|--------------|-------|
| Run avec le compagnon | 10-30 | Selon étage atteint |
| Quête personnelle | 50-200 | Quand disponible |
| Shadow Vigil | 5-15 | Par session |
| Cadeau de Résonance | 20-50 | Items spéciaux rares |
| Événement saisonnier | 30-100 | Bonus event |

## 4. Echo Fragments

### Description
Les **Echo Fragments** sont des items équipables obtenus via le système de Résonance. Chaque compagnon a un Echo Fragment unique reflétant sa personnalité et ses pouvoirs.

### Tiers

| Tier | Débloqué à | Type d’effet | Upgradeable |
|------|------------|-------------|-------------|
| **T1** | Résonance L3 | Effet passif uniquement | Oui (Resonance Crystals) |
| **T2** | Résonance L12 | Effet passif + effet actif | Oui (Resonance Crystals) |

### Upgrade avec Resonance Crystals
- Chaque Echo Fragment a 5 niveaux d’upgrade (I à V)
- Le coût en **Resonance Crystals** augmente par niveau
- Chaque niveau améliore l’effet passif de +20%
- Au niveau V, l’effet est 2x plus puissant qu’au niveau I

| Niveau | Coût (Resonance Crystals) | Bonus |
|--------|---------------------------|-------|
| I | 0 (base) | Effet de base |
| II | 10 | +20% efficacité |
| III | 20 | +40% efficacité |
| IV | 35 | +70% efficacité |
| V | 50 | +100% efficacité |

### Exemples d’Echo Fragments

| Compagnon | Fragment | Effet T1 | Effet T2 (actif) |
|-----------|----------|----------|-------------------|
| Kaelen | Écho de l’Épée | +5% Crit Rate en combat | Frappe spectrale (dmg burst, CD 30s) |
| Lyra | Écho du Chant | +3% HP régen/s hors combat | Chant de guérison (heal AoE, CD 45s) |
| Sable | Écho de l’Ombre | +8% dmg depuis stealth | Téléportation d’ombre (blink, CD 20s) |
| Ignis | Écho de la Flamme | +6% Fire DMG | Explosion nova (AoE feu, CD 35s) |
| Nyx | Écho du Void | +4% dégâts Néant | Fracture dimensionnelle (stun AoE, CD 40s) |

## 5. Battle Pass (Issue #103)

### Structure

| Paramètre | Valeur |
|-----------|--------|
| **Durée** | 1 saison (10 semaines) |
| **Tiers totaux** | 100 |
| **Tracks** | Gratuit + Premium |
| **XP par tier** | 1000 XP (constant) |

### Sources d’XP Battle Pass

| Source | XP | Fréquence |
|--------|-----|-----------|
| Quête quotidienne | 100 | 3/jour |
| Défi hebdomadaire | 500 | 3/semaine |
| Défi saisonnier | 1000 | 5 par saison |
| Run complété | 50 | Illimité |
| Boss kill | 25 | Illimité |

### Récompenses (exemples)

| Tier | Track Gratuit | Track Premium |
|------|--------------|---------------|
| 1 | 500 Cendres | Cosmetic : cadre de profil |
| 10 | 10 Void Shards | Skin de compagnon |
| 25 | Rune Épique aléatoire | 50 Void Shards |
| 50 | 30 Void Shards | Skin d’arme |
| 75 | Ticket d’Arena x5 | 100 Void Shards + Resonance Crystal x5 |
| 100 | Titre « Sentinelle » | Cosmetic légendaire + 200 Void Shards + Essence Antique x1 |

### Progression typique
- **Joueur casual** (30 min/jour) : ~70 tiers en 10 semaines
- **Joueur régulier** (1h/jour) : ~90 tiers en 10 semaines
- **Joueur hardcore** (2h+/jour) : 100 tiers en 6-7 semaines

## 6. Réputation de Clan

### Rangs de Clan

| Rang | Réputation requise | Avantages |
|------|---------------------|-----------|
| **Bronze** | 0-999 | Accès aux contrats de base, chat de clan |
| **Argent** | 1000-2999 | Bonus XP +5%, boutique de clan T1 |
| **Or** | 3000-5999 | Bonus XP +10%, boutique T2, Abyssal Dust hebdo |
| **Obsidien** | 6000-9999 | Bonus XP +15%, boutique T3, cosmetics de clan |
| **Légendaire** | 10000+ | Bonus XP +20%, boutique T4, titre exclusif, aura de clan |

### Gagner de la Réputation

| Activité | Réputation | Fréquence |
|----------|-----------|-----------|
| Contrat de clan complété | 50-200 | Hebdomadaire (pool commun) |
| Don de ressources au clan | 10-50 | Quotidien (cap) |
| Participation au raid de clan | 100-300 | Hebdomadaire |
| Victoire Arena pour le clan | 5-15 | Par victoire |
| Événement saisonnier (clan) | 200-500 | Par événement |

### Déclassement
- La réputation décroît de **-50/semaine** si le joueur est inactif (pas de connexion pendant 7 jours)
- Minimum : ne descend jamais en dessous du seuil du rang inférieur + 100
- Réactivation : 1 semaine d’activité stoppe la décroissance

## 7. Progression en Run

### Étages et Difficulté

| Étages | Zone | Corruption | Ennemis |
|--------|------|------------|---------|
| 1-3 | Cendres Désolées | 0-30% | Tier I-II |
| 4-6 | Cryptes d’Ombre / Profondeurs Carmines | 30-60% | Tier II-III |
| 7-9 | Faille du Néant | 60-90% | Tier III-IV |
| 10-12 | Coeur du Void | 90-120% | Tier IV + Boss final |

### Choix post-Boss

Après chaque boss vaincu (tous les 3 étages) :

| Option | Effet |
|--------|-------|
| **Bénédiction** | Buff permanent pour le run (+ATK, +HP, +Drop Rate, etc.) |
| **Malédiction** | Débuff mais récompenses x1.5 pour les étages suivants |
| **Sacrifice** | Perdre un buff existant contre une rune rare garantie |
| **Skip** | Aucun changement, sécurité maximale |

## 8. Synthèse des Couches de Progression

```
Niveau Joueur (1-60)
  └── Débloque systèmes et zones
  └── Arbre de Talents (15 nodes)
       └── Offense / Défense / Utilité

Compagnon
  └── Affinité (0-100, 5 seuils)
  └── Résonance (1-15, Echo Fragments + Void Form)
  └── Préférence Relationnelle (Romance / Amitié / Neutre)

Équipement
  └── Corruption Runes (7 sets, 6 slots, +15)
  └── Void Forge (Reforging, Awakening, Infusion)

Social
  └── Clan (Bronze → Légendaire)
  └── Void Arena (5 tiers PvP)
  └── Battle Pass (100 tiers/saison)

Roguelite
  └── Bénédictions / Malédictions par run
  └── Corruption croissante
  └── Loot temporaire → Extraction
```
